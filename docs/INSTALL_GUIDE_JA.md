# MaskingTool インストールガイド（Windows / macOS）

このガイドは「ターミナルを使わないユーザー」向けです。  
`npm` や `git` の操作は不要です。

## 1. ダウンロード

1. [MaskTool Releases](https://github.com/gatuwo/MaskTool/releases) を開く
2. 最新リリースの `Assets` を開く
3. 自分のOSに合ったファイルをダウンロード

Windows の場合:

- `MaskingTool-win-...-x64.exe`（インストーラ）
- または `MaskingTool-win-...-x64.zip`

macOS の場合:

- `MaskingTool-mac-...-arm64.dmg` または `MaskingTool-mac-...-x64.dmg`
- または `MaskingTool-mac-...zip`

## 2. インストール

### Windows

1. `*.exe` をダブルクリック
2. 「WindowsによってPCが保護されました」が出る場合:
3. `詳細情報` → `実行`
4. 画面の案内に沿ってインストール

### macOS

1. `*.dmg` を開く
2. `MaskingTool.app` を `Applications` にドラッグ
3. 初回起動時に警告が出る場合:
4. Finderでアプリを右クリック → `開く` → `開く`

## 3. 使い方（最初にやること）

1. アプリを起動
2. `画像を選択` で画像を読み込み
3. モザイクまたは黒ベタを選んで編集
4. `フォルダ選択` を押して保存先を選ぶ
5. `PNG出力` で保存

## 4. 操作メモ

- macOS: `Command + マウスホイール` でズーム
- Windows: `Ctrl + マウスホイール` でズーム

## 5. 更新方法

1. Releases から新しいバージョンをダウンロード
2. 旧版を終了
3. 新版を上書きインストール（Windows）または差し替え（macOS）
