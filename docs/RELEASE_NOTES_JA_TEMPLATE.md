### 対応OS

- Windows (x64)

### ダウンロード

- Windows: `MaskingTool-win-...-x64.exe`（推奨）

- macOS: `MaskingTool-mac-...-arm64.dmg` / `MaskingTool-mac-...-x64.dmg`
